# 상품 관리 기능 테스트 방법

## 변경된 내용
- ✅ 모든 카테고리 상품 표시 (기존: 12개만)
- ✅ 성별/카테고리 필터링 기능 추가
- ✅ 모든 상품 수정 가능

## 테스트 방법

### 1. 개발 서버 실행
```bash
npm run dev:store
```
또는 Git Bash에서:
```bash
cd /c/Users/dladu/OneDrive/바탕\ 화면/IM/areum
npm run dev:store
```

### 2. 브라우저에서 확인
- http://localhost:3001 접속
- 관리자 로그인 (areum1004@gmail.com / areum1004!)
- 관리자 대시보드 → **상품 관리** 탭 클릭

### 3. 확인할 사항
- ✅ 상단에 필터 버튼이 보이는지 (성별: 전체/여성/남성, 카테고리: 전체/아우터/셔츠...)
- ✅ 필터 변경 시 해당하는 상품들이 표시되는지
- ✅ 모든 카테고리의 상품이 표시되는지 (여성/아우터만이 아니라)
- ✅ 각 상품의 "수정" 버튼 클릭 시 모달이 뜨는지
- ✅ 모달에서 제품명, 가격, 이미지 모두 수정 가능한지

### 4. 캐시 문제 해결
변경사항이 보이지 않으면:
- **브라우저 강력 새로고침**: `Ctrl+Shift+R` (Windows) 또는 `Cmd+Shift+R` (Mac)
- 또는 개발자 도구(F12) → Network 탭 → "Disable cache" 체크

## 문제 해결

### 필터가 작동하지 않는 경우
- 브라우저 콘솔(F12)에서 오류 확인
- 개발 서버 재시작

### 모든 상품이 안 보이는 경우
- 필터에서 "전체" 선택 확인
- 페이지 새로고침
